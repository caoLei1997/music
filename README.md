# React-Music
### 巩固技能，练习技能
### 使用技术 
1. react18
2. router6
3. vite
4. styled-component
5. Immutable


### 依赖 node16、 pnpm@7x

### 启动步骤
1. git clone
2. pnpm i
3. git submodule update --init --recursive
4. npm run submodules
5. npm run start
